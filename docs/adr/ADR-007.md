# ADR-007: Defer QuickSight from the Active Runtime

- Short Name: `defer-quicksight`
- Status: Accepted

## Context

QuickSight is a reasonable future integration path, but it introduces managed BI service costs and is not required to demonstrate the lakehouse pipeline itself.

## Decision

Exclude QuickSight from the active runtime path and treat it as a future feature.

## Consequences

### Positive

- Lower demo cost
- Simpler architecture story
- Focus remains on the ETL pipeline and curated outputs

### Negative

- No managed in-cloud dashboarding in the current implementation

## Architecture Impact

The architecture remains BI-ready, but QuickSight is not provisioned as part of the live project scope.
