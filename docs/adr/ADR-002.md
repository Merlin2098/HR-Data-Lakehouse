# ADR-002: Explicit Glue Catalog Metadata

- Short Name: `explicit-glue-catalog-metadata`
- Status: Implemented

## Context

Silver and gold datasets have controlled schemas defined by contracts and Terraform. The project needs Athena-readable metadata without relying on schema inference.

## Decision

Use AWS Glue Data Catalog as the metadata layer for silver and gold, without using Glue Crawlers.

## Alternatives Considered

- Glue Crawlers
  - More automated
  - Lower schema control
  - Higher risk of drift and unwanted inference changes

## Consequences

### Positive

- Full schema control
- Better alignment with contracts
- More predictable Athena behavior

### Negative

- Higher upfront modeling effort
- Manual maintenance when schemas evolve

## Architecture Impact

The catalog acts as an explicit governance layer rather than an automatic discovery mechanism.
